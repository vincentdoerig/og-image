What is this?

This is a service that generates dynamic Open Graph images that I can embed in the <meta> tags of my sites likes doerig.dev or tools.doerig.dev.

For each keystroke, headless chromium is used to render an HTML page and take a screenshot of the result which gets cached.

Find out how this works and deploy your own image generator by visiting GitHub.
